# DataWhisperer v1 发布说明

发布时间：2026-06-02

## 版本定位

v1 是 DataWhisperer 的第一个可运行版本，目标是跑通 Text-to-SQL 数据分析智能体的最小闭环。

当前版本只做后端 API 和中文控制台，不做复杂权限、企业级数据源管理和多智能体拆分。

## 已完成功能

- FastAPI 后端服务。
- 中文 Web 控制台。
- MySQL 电商销售示例库。
- 数据库 schema 自动读取。
- OpenAI-compatible 大模型客户端。
- 自然语言生成 SQL。
- SQL 只读安全校验。
- 查询执行和异常处理。
- ECharts 图表配置推荐。
- 查询结果业务分析。
- Agent 执行轨迹返回。
- 示例问题接口。
- pytest 和 ruff 基础质量检查。

## 主要接口

- `GET /api/health`
- `GET /api/examples`
- `GET /api/schema/overview`
- `POST /api/chat/query`

## 当前限制

- 只支持 MySQL 示例库。
- 只支持只读查询。
- 图表推荐规则偏轻量。
- 分析结论仍依赖大模型质量。
- 暂未支持用户登录、权限控制和多数据源管理。

## 适合演示的问题

- 查询最近 6 个月每月销售额趋势
- 查询各商品品类销售额占比
- 哪个地区客单价最高
- 找出销售额下滑最明显的商品
- 查询华东地区销量前三的商品及其环比增长

## 下一版本建议

v1.1 可以继续增强 GitHub 展示和工程质量：

- 增加运行截图。
- 增加 API 调用示例。
- 增加更多测试。
- 增加 GitHub Actions。

v2 可以进入提示词治理：

- 把 prompt 独立成模板文件。
- 给 prompt 增加版本号。
- 增加模型输出解析和失败重试。

v3 可以加入 RAG 指标口径库：

- 维护 GMV、客单价、复购率等指标定义。
- 让模型先检索指标口径，再生成 SQL。

v4 可以做 MCP 工具化：

- 将 schema、SQL、查询、图表、分析能力包装成 MCP 工具。
- 让其他 Agent 或客户端复用这些工具。
